🎯 3 TOMBOL PENTING SETELAH COMMIT:

Di Source Control panel, klik "..." (3 titik), akan muncul menu:

    Pull, Push...
    Sync
    ───────────────
    Fetch
    Pull
    Pull (Rebase)
    Push                  ← Klik ini untuk upload ke GitHub!
    Push to...
    ───────────────
    Remote                ← Klik ini untuk add remote GitHub
      > Add Remote...     ← Pilih ini, paste URL GitHub
      > Remove Remote
    ───────────────
    Branch...
    Checkout to...
    ───────────────
    Tags                  ← (Opsional) untuk versioning
      > Create Tag...
      > Delete Tag...

✅ FILE YANG HARUS ADA (di Changes list):

    ✅ src/              → Source code TypeScript
    ✅ views/            → Template EJS
    ✅ public/           → CSS, JS, images
    ✅ package.json      → Dependencies
    ✅ tsconfig.json     → TypeScript config
    ✅ .gitignore        → File proteksi
    ✅ README.md         → Dokumentasi
    ✅ ecosystem.config.js → PM2 config

═══════════════════════════════════════════════════════════════════

❌ FILE YANG JANGAN SAMPAI ADA:

    ❌ .env              → PASSWORD DATABASE! BAHAYA!
    ❌ node_modules/     → Terlalu besar (ratusan MB)
    ❌ dist/             → Compiled code (bisa di-generate)
    ❌ uploads/          → File upload user
    ❌ backups/          → Backup database
    ❌ whatsapp-session/ → Session WhatsApp
    ❌ logs/*.log        → Log files

    Kalau file di atas muncul = STOP! Ada yang salah!
