CARA MEMBUAT FILE EXCEL UNTUK IMPORT PELANGGAN
==============================================

LANGKAH 1: BUKA EXCEL BARU
--------------------------
- Buka Microsoft Excel
- File > New > Blank Workbook

LANGKAH 2: ISI HEADER (BARIS 1)
--------------------------------
Cell A1: Nama
Cell B1: Telepon  
Cell C1: Alamat

PENTING: Ketik PERSIS seperti di atas (huruf besar kecil harus sama!)

LANGKAH 3: ISI DATA (MULAI BARIS 2)
------------------------------------
Cell A2: Test User 1
Cell B2: 081999999991
Cell C2: Jl. Test 1

Cell A3: Test User 2
Cell B3: 081999999992
Cell C3: Jl. Test 2

LANGKAH 4: SAVE FILE
--------------------
- File > Save As
- Pilih lokasi: Desktop
- File name: test_import.xlsx
- Save as type: Excel Workbook (*.xlsx)
- Klik Save

LANGKAH 5: IMPORT
-----------------
- Buka aplikasi billing
- Pelanggan > Data Pelanggan
- Klik Import Excel
- Pilih file test_import.xlsx
- Klik Import Data

HARUSNYA: Berhasil: 2, Gagal: 0

==============================================

JIKA MASIH GAGAL:
1. Screenshot error yang muncul
2. Cek terminal server - lihat ada error merah atau tidak
3. Restart server dengan: pm2 restart billing-system
