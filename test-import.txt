CARA TEST IMPORT EXCEL - LANGKAH MUDAH
=====================================================

STEP 1: RESTART SERVER
-----------------------
Klik 2x file: restart-import.bat
(File ini ada di folder yang sama dengan file ini)

Tunggu sampai muncul tulisan "SERVER SUDAH RESTART!"


STEP 2: BUAT FILE EXCEL
------------------------
1. Buka Excel BARU (blank workbook)

2. Ketik di baris pertama (HEADER):
   Cell A1: Nama
   Cell B1: Telepon
   Cell C1: Alamat

3. Ketik di baris kedua (DATA):
   Cell A2: Test User
   Cell B2: 081999999999
   Cell C2: Jl. Test

4. Save As > Desktop > test_import.xlsx


STEP 3: IMPORT FILE
-------------------
1. Buka browser: http://localhost:3000
2. Tekan Ctrl + Shift + R (refresh keras)
3. Menu: Pelanggan > Data Pelanggan
4. Klik: Import Excel
5. Pilih file: test_import.xlsx
6. Klik: Import Data
