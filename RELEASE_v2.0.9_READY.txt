🚀 DEPLOY TO PRODUCTION
════════════════════════════════════════════════════════════════════════════

OPTION 1: One-Line Deploy (Copy-Paste)
────────────────────────────────────────────────────────────────────────────
Ganti YOUR_IP dengan IP server Anda:

ssh root@YOUR_IP "cd /opt/billing && git pull origin main && npm run build && pm2 restart billing-app && pm2 list"


OPTION 2: Use Auto-Deploy Script
────────────────────────────────────────────────────────────────────────────
1. Double-click: DEPLOY-SEKARANG.bat
2. Edit IP di baris 11: set SERVER_IP=YOUR_IP
3. Save & double-click


OPTION 3: Manual Step-by-Step
────────────────────────────────────────────────────────────────────────────
ssh root@YOUR_IP
cd /opt/billing
git pull origin main
npm run build
pm2 restart billing-app
pm2 list

📋 TEST IMPORT AFTER DEPLOY
════════════════════════════════════════════════════════════════════════════

1. Buat Excel dengan 3 kolom:

   Nama          | Telepon      | Alamat
   Test User     | 08999999999  | Jl. Test No. 1
   John Doe      | 08123456789  | Jl. Merdeka 123


2. Upload di browser:
   http://YOUR_IP/customers/list → "📥 Import Excel"


3. Monitor log (terminal lain):
   ssh root@YOUR_IP "pm2 logs billing-app --lines 0"


4. Expected result:
   ✅ Berhasil: 2, Gagal: 0

🔧 TROUBLESHOOTING
════════════════════════════════════════════════════════════════════════════

If deploy fails:
────────────────────────────────────────────────────────────────────────────
ssh root@YOUR_IP
cd /opt/billing
npm install
npm run build
pm2 restart billing-app


If import fails:
────────────────────────────────────────────────────────────────────────────
1. Run: CHECK-IMPORT-ERROR.ps1
2. Check PM2 logs: pm2 logs billing-app --lines 50
3. Verify Excel header: Nama | Telepon | Alamat (Indonesian!)
