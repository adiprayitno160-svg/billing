üé® Upload Pakai Cursor (INI YANG PENTING!)

A. Buka Source Control

  • Tekan Ctrl + Shift + G
  • Atau klik icon cabang pohon (‚éá) di sidebar kiri

B. Initialize Git

  • Kalau ada button "Initialize Repository" ‚Üí klik!
  • Kalau tidak ada ‚Üí sudah ter-init, lanjut

C. Cek File (SANGAT PENTING!)

‚úÖ Yang HARUS ADA:
  • src/ (source code)
  • views/ (templates)
  • public/ (static files)
  • package.json
‚ùå Yang JANGAN SAMPAI ADA:
  • .env ‚Üê BAHAYA! Berisi password database!
  • node_modules/
  • backups/
  • uploads/

‚ö†Ô∏è KALAU .env MUNCUL = STOP! JANGAN LANJUT!

D. Stage Files

  • Klik tombol "+" di samping tulisan "Changes"
  • File akan pindah ke "Staged Changes"

E. Commit

  1. Ketik message di kotak atas:
Initial commit - Billing System v1.0.0
  1. Klik tombol ‚úì Commit atau tekan Ctrl + Enter

F. Add Remote (Sambung ke GitHub)

  1. Klik "..." (3 titik) di pojok kanan atas panel
  2. Pilih: Remote ‚Üí Add Remote
  3. Paste URL GitHub (dari Step 1)
  4. Enter
  5. Nama: origin (default)

G. Push (Upload!)

  1. Klik "..." lagi
  2. Pilih: Push
  3. Tunggu proses upload (10-60 detik)
  4. Kalau diminta login ‚Üí login dengan akun GitHub
‚úÖ SELESAI! Kode sudah di GitHub!

üÜò Troubleshooting
